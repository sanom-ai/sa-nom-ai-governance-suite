# Support Surface

This folder contains non-runtime material that supports development without crowding the project root.

## Subfolders

- `tests/`: active regression suite
- `examples/`: reusable example payloads and fixtures
- `dev_notes/`: round notes, closeout docs, roadmaps, and structure audits
- `specs/`: working specs and architecture notes used during development
- `review_tools/`: tools for building review packs and other support automation
- `context_bundle/`: imported external context kept for comparison or review only
- `legacy_*` and `tmp_*`: archival or temporary folders that are not live product code
