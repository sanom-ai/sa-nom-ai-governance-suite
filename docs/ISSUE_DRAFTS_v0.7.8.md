# ISSUE DRAFTS v0.7.8

## Slice 1 - Tablet Command Surface Foundation
- Rework the command shell so tablet users see Home first and the command dock becomes compact, secondary, and touch-friendly.
- Make the top navigation horizontally usable on tablet without turning into a wrapped desktop menu.
- Increase touch target quality for navigation, organization switching, and command actions.
- Prioritize department quick access by active work context so limited tablet space shows the most useful teams first.

## Slice 2 - Private Session Continuity
- Improve session continuity and refresh posture for private tablet lanes.
- Make reconnect, idle, and sign-out behavior clearer for operator-managed devices.

## Slice 3 - Role-Based Tablet Views
- Keep one shell, but bias the default experience by persona focus.
- Preserve Control Room as an advanced path for admin, IT, and founder roles only.

## Slice 4 - Touch-First Work Flows
- Tighten touch flows for Work Inbox, Cases, Documents, AI Actions, and human approvals.
- Reduce unnecessary taps between command-surface summary and the next governed action.
