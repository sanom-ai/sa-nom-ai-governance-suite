# Commercial License & Pricing

SA-NOM AI Governance Suite is open source under AGPL-3.0-only.
The commercial lane is for organizations that want production-grade governance API operations with recurring assurance and clear platform accountability.

## Commercial Positioning

SA-NOM commercial is API product-first.

- Default mode: self-serve governance API
- Customer controls policy and approval intent
- SA-NOM operates platform quality (`uptime`, `security`, `release`, `billing`)
- No default managed-service dependency

For product boundaries, see:
- [PRODUCT_SCOPE.md](PRODUCT_SCOPE.md)
- [CUSTOMER_PLATFORM_RESPONSIBILITY_MATRIX.md](CUSTOMER_PLATFORM_RESPONSIBILITY_MATRIX.md)
- [COMMERCIAL_BOUNDARY.md](COMMERCIAL_BOUNDARY.md)

## What Commercial Includes

Commercial scope covers production use of SA-NOM as a governance runtime API platform:

- private-first governed runtime operations
- tenant isolation, API access control, and usage governance
- policy decision enforcement and auditable dispatch flow
- evidence and audit export posture
- release and runtime confidence governance
- subscription and usage-based commercial operation

Commercial scope does not change the product boundary:

- SA-NOM does not own customer policy decisions
- SA-NOM is not a daily case-handling team
- SA-NOM is not an unlimited support desk by default

## Pricing (2026 baseline)

All prices are annual commercial bands unless otherwise quoted.

| Tier | Price (THB) | Best fit | Product posture |
| --- | --- | --- | --- |
| **Community** | **Free** | Self-managed evaluation and internal learning | Open-source baseline |
| **Governance API Launch** | **120,000 - 250,000 / year** | Teams starting controlled API governance in one environment | Subscription baseline, self-serve |
| **Governance API Production** | **950,000 - 1,800,000 / year** | Production teams with serious workflow and audit needs | Multi-lane runtime with stronger controls |
| **Governance API Regulated** | **1,200,000 - 3,600,000 / year** | Regulated or higher-assurance environments | Hardening, evidence depth, stricter governance posture |
| **Enterprise / Sovereign / Gov** | **Quote only** | Large, sovereign, or heavily regulated deployments | Quote-shaped, selective acceptance |

For high-complexity environments, pricing may extend above the listed bands based on scope and regulatory posture.

## Optional Add-Ons (Bounded)

Add-ons are optional and scope-defined. They do not change the API product-first model.

## Recurring Revenue Model

SA-NOM commercial is designed as recurring platform revenue:

- annual subscription by tier
- optional usage component by runtime volume
- annual renewal with release-confidence and security posture continuity
